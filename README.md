## Project Overview

This project is a full-stack To-Do List application with CRUD functionality, containerized using Docker and deployed on Render.com. The application demonstrates the implementation of Continuous Integration and Continuous Deployment (CI/CD) principles.

### Live Application URLs
- **Frontend:** [https://fe-todo.onrender.com](https://fe-todo.onrender.com)
- **Backend API:** [https://be-todo.onrender.com](https://be-todo.onrender.com)
- **Backend Health Check:** [https://be-todo.onrender.com/health](https://be-todo.onrender.com/health)

### Technology Stack
| Component | Technology |
|-----------|------------|
| **Frontend** | React.js |
| **Backend** | Node.js + Express |
| **Database** | PostgreSQL |
| **Containerization** | Docker |
| **Container Registry** | Docker Hub |
| **Cloud Deployment** | Render.com |
| **Version Control** | Git & GitHub |

---

## Application Features

✅ **Create Tasks** - Add new tasks with title and description  
✅ **View Tasks** - Display all tasks in a clean, organized list  
✅ **Edit Tasks** - Update task details or description  
✅ **Delete Tasks** - Remove tasks when completed  
✅ **Mark Complete** - Toggle task completion status  
✅ **Responsive Design** - Works on desktop and mobile devices  
✅ **Persistent Storage** - All tasks saved in PostgreSQL database
